URL Shortener Microservice


How to Use:

Pass a valid URL as a parameter.

In return, you will get a JSON response that shows your original URL and a shortened version.

If you paste the shortened URL into the URL box, then it will take you to your original URL destination.


Example:

url/http://www.amazon.com
url/http://www.github.com

Example Output:

{"Normal_URL":"http://www.amazon.com","Shortened_URL":7108}
